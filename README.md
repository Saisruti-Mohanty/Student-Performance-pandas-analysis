# Student Performance Analysis using Pandas

## 📊 Project Overview

This project analyzes student performance data using Python and the Pandas library.
The goal is to explore patterns in students' scores and understand how different factors affect academic performance.

## 🛠 Tools & Technologies

* Python
* Pandas
* Jupyter Notebook

## 📁 Dataset

The dataset contains student exam scores and background information such as:

* Gender
* Parental level of education
* Lunch type
* Test preparation course
* Scores in math, reading, and writing

## 🔍 Analysis Performed

* Data loading and exploration
* Checking dataset structure
* Data summary statistics
* Basic performance analysis using Pandas functions

## 📂 Project Files

* **StudentsPerformance.csv** – Dataset used for analysis
* **STUDENT_PERFORMANCE_ANALYSIS.ipynb** – Jupyter Notebook with Pandas analysis
* **README.md** – Project documentation
